## 🔍 问题描述

最近 **Windows** 又给我强制更新了，更新之后发现每次重启电脑之后，所有的应用都需要重新登录，包括：

- 🌐 浏览器
- 💬 聊天软件
- ⚙️ 甚至一些系统应用

怎么这么可恶。这个问题一共在我的印象里面出现了**两次**。

---

## 💡 解决方案

### 第一次尝试

第一次出现这个问题的时候，我直接用了 `sfc /scannow` 命令，是的直接在**管理员的终端**里面运行这个命令，等它扫描完修复完之后，重启电脑，问题就解决了。

**操作步骤：**

```bash
sfc /scannow
```

::: info 
`sfc /scannow` 命令是 **Windows系统文件检查器**（System File Checker）的一个组件，它用于扫描所有受保护的系统文件，并替换损坏的文件。当您的系统出现不稳定、错误消息或其他异常行为时，运行 `sfc /scannow` 命令可能有助于解决问题。
:::

---

### 第二次尝试

这次意外地发现 `sfc /scannow` 命令并没有解决问题。然后我就去试试 `dism /online /cleanup-image /restorehealth` 命令。是的，也是在终端输入这个命令。但是老是在 **62.3%** 的时候就卡住了，非常离谱，而且一直吃不上代理。

**操作步骤：**

```bash
dism /online /cleanup-image /restorehealth
```

后面把电脑多重启了几遍，明明没有更新并重启，但是在有一次重启的时候他显示更新了，然后就好了，也是非常的离谱。

::: info 
**DISM**（部署映像服务和管理工具）是一个强大的命令行工具，用于修复 Windows 系统映像。当系统出现问题时，可以使用 `dism /online /cleanup-image /restorehealth` 命令来修复系统映像。
:::

---

## 📝 总结

遇到 Windows 更新后需要重新登录的问题，可以尝试以下方法：

1. **首选方案**：运行 `sfc /scannow` 修复系统文件
2. **备用方案**：运行 `dism /online /cleanup-image /restorehealth` 修复系统映像
3. **最后手段**：多次重启电脑，等待系统自动修复
